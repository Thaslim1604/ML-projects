Welcome to my **Machine Learning & AI Projects** repository! This space contains a curated collection of machine learning, deep learning, and data science projects built for **learning, academic coursework, and portfolio purposes**.

Each project explores a different facet of AI — from predictive modeling and NLP to computer vision and intelligent document understanding — using tools like **Python, Scikit-learn, TensorFlow/PyTorch, Hugging Face Transformers, Flask, and Streamlit**.
